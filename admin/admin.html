CPU

The central processing unit (CPU) is the brain of the computer.
Motherboard

The motherboard connects all the components of the computer.
Memory

Memory (RAM) is used for temporary data storage and quick access.
Storage

Storage devices like SSDs and HDDs store data permanently.
Case

The case houses all the components of the computer.
PSU

The power supply unit (PSU) provides power to the components.
GPU

The graphics processing unit (GPU) handles rendering visuals.
